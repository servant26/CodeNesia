Tentang Materi

Course CodeIgniter ini dirancang untuk membantumu memahami framework PHP yang sederhana namun powerful. Kamu akan belajar membuat fitur CRUD, mengatur template view, serta menghubungkan aplikasi dengan database.

Cocok untuk developer yang ingin membuat aplikasi web dinamis dengan framework ringan, mudah dipelajari, dan efisien dalam pengembangan.

Apa yang akan kamu pelajari?


  • Memahami konsep dasar dan arsitektur MVC yang digunakan dalam framework CodeIgniter
  • Mampu menyiapkan, mengonfigurasi, dan menjalankan proyek CodeIgniter dengan benar
  • Menguasai alur kerja routing, controller, model, dan view untuk membangun aplikasi web dinamis
  • Mampu mengelola database, membuat fitur CRUD, serta menangani form dan input pengguna
  • Menangani autentikasi, session, validasi, dan keamanan aplikasi dengan fitur bawaan CodeIgniter
  • Memanfaatkan helpers, libraries, serta fitur tambahan seperti upload file, pagination, dan search
  • Mampu melakukan debugging, error handling, dan mengoptimalkan performa aplikasi
  • Membangun satu proyek web lengkap dengan CodeIgniter sebagai penerapan seluruh konsep yang telah dipelajari
