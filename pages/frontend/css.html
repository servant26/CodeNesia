Tentang Materi

Course CSS ini dirancang untuk membantumu menguasai teknik styling pada website. Kamu akan belajar mulai dari dasar penulisan CSS, pengaturan tata letak dengan Flexbox dan Grid, hingga penggunaan animasi dan responsive design.

Cocok untuk pemula yang ingin meningkatkan keterampilan front-end atau siapa pun yang ingin membuat tampilan web lebih menarik dan modern.

Apa yang akan kamu pelajari?


  • Memahami dasar penulisan CSS serta cara menyisipkannya ke dalam dokumen HTML
  • Mengenal berbagai jenis selector, spesifisitas, dan cara mengatur prioritas styling
  • Mampu mengatur warna, teks, font, background, dan border untuk mempercantik tampilan web
  • Mengetahui konsep box model, ukuran elemen, dan properti display untuk mengontrol layout
  • Menguasai teknik layout modern menggunakan Flexbox dan CSS Grid
  • Membuat tampilan yang responsif di berbagai ukuran layar menggunakan media query
  • Menambahkan efek interaktif dengan transition, animation, dan transform
  • Memahami penggunaan pseudo-class dan pseudo-element untuk styling dinamis
  • Membangun satu proyek web sederhana sebagai penerapan seluruh konsep CSS yang telah dipelajari
