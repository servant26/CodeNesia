Tentang Materi

Course Git ini dirancang untuk membantumu memahami alur kerja pengembangan software yang profesional. Kamu akan belajar membuat repository, mengelola perubahan, menangani konflik, hingga berkolaborasi melalui platform seperti GitHub.

Cocok untuk developer, designer, atau siapa pun yang ingin bekerja secara tim dengan manajemen kode yang teratur, aman, dan mudah dilacak.

Apa yang akan kamu pelajari?


  • Memahami apa itu Git dan peran pentingnya dalam version control
  • Mengetahui perbedaan Git dengan platform hosting seperti GitHub atau GitLab
  • Menginstal dan mengonfigurasi Git di berbagai sistem operasi
  • Menguasai alur kerja dasar Git: inisialisasi, staging, commit, dan log
  • Bekerja dengan branch untuk mengelola fitur dan melakukan merge antar cabang
  • Menghubungkan repository lokal dengan remote seperti GitHub
  • Berlatih kolaborasi tim melalui fork, pull request, dan workflow industri
  • Menjelajahi riwayat commit, melihat perubahan, dan memperbaiki kesalahan
  • Menyimpan perubahan sementara menggunakan stash
  • Menerapkan best practice seperti pesan commit yang baik dan struktur branch rapi
  • Membuat tag versi dan mengelola rilis proyek dengan profesional
