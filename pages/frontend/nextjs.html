Tentang Materi

Course Next.js ini dirancang untuk membantumu menguasai framework React yang powerful untuk pengembangan web profesional. Kamu akan belajar tentang file-based routing, data fetching, API routes, hingga proses deployment ke production.

Cocok untuk developer yang ingin membawa skill React ke level berikutnya dan membuat aplikasi web yang cepat, teroptimasi, dan mudah dikelola.

Apa yang akan kamu pelajari?


  • Memahami konsep dasar dan struktur project Next.js
  • Membuat halaman, routing, dan navigasi dinamis
  • Mengambil data dari API dan menampilkannya di halaman
  • Menerapkan styling menggunakan Tailwind CSS
  • Mengelola state lokal dan global pada aplikasi
  • Menerapkan autentikasi dan proteksi halaman
  • Menghubungkan aplikasi dengan database menggunakan Prisma
  • Menerapkan praktik optimasi dan deployment ke Vercel
  • Membangun proyek akhir yang menggabungkan seluruh konsep
