Tentang Materi

Course Python ini dirancang untuk membantumu menguasai bahasa pemrograman serbaguna yang mudah dipelajari. Kamu akan belajar membuat program otomatisasi, analisis data, hingga dasar pengembangan web dan machine learning.

Cocok untuk pemula yang ingin mulai belajar coding, maupun developer yang ingin menambah skill dengan bahasa yang fleksibel dan banyak digunakan di dunia industri.

Apa yang akan kamu kuasai setelah mempelajari ini?


  • Memahami konsep dasar dan filosofi Python sebagai bahasa pemrograman yang sederhana dan powerful
  • Menyiapkan lingkungan pengembangan serta menjalankan program Python di berbagai platform
  • Menguasai logika dasar pemrograman seperti variabel, operator, percabangan, dan perulangan
  • Mampu mengelola data menggunakan struktur seperti list, tuple, set, dan dictionary
  • Menulis fungsi untuk membuat kode yang lebih efisien, modular, dan mudah dibaca
  • Memahami konsep OOP (Object-Oriented Programming) termasuk class, object, inheritance, dan polymorphism
  • Membangun proyek Python sederhana yang menerapkan seluruh konsep dasar pemrograman secara terintegrasi
