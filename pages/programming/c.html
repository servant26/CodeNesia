Tentang Materi

Course C ini dirancang untuk membantumu memahami cara kerja program dari tingkat dasar. Kamu akan belajar konsep memory management, array, pointer, dan struktur data sederhana yang sering digunakan dalam pengembangan software.

Cocok untuk pemula yang ingin membangun dasar logika kuat sebelum mempelajari bahasa lain, atau siapa pun yang ingin memahami cara kerja program secara mendalam dan efisien.

Apa yang akan kamu pelajari?


  • Memahami sejarah, karakteristik, dan dasar-dasar bahasa pemrograman C sebagai fondasi bahasa modern lainnya
  • Mampu menulis, menjalankan, dan mengelola program C dari awal menggunakan compiler dan lingkungan pengembangan
  • Menguasai sintaks dasar, struktur program, serta kontrol alur seperti percabangan dan perulangan
  • Mampu menggunakan array, string, dan fungsi untuk mengorganisir kode dengan efisien
  • Memahami konsep pointer dan penerapannya dalam pengelolaan data serta memori
  • Mempelajari struktur data dasar, file handling, serta pengalokasian memori secara dinamis
  • Menangani kesalahan dan melakukan debugging agar program berjalan stabil dan bebas error
  • Membangun satu proyek akhir menggunakan bahasa C sebagai penerapan dari seluruh konsep yang telah dipelajari
