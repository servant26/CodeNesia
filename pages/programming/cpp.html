Tentang Materi

Course C++ ini dirancang untuk membantumu menguasai konsep OOP dan efisiensi pemrograman tingkat sistem. Kamu akan belajar penggunaan pointer, fungsi, struktur data, hingga manipulasi memori yang lebih dalam.

Cocok untuk developer yang ingin memperkuat kemampuan logika, membangun aplikasi performa tinggi, atau melanjutkan karier di bidang software development dan game programming.

Apa yang akan kamu pelajari?


  • Memahami konsep dasar bahasa pemrograman C++ serta perbedaannya dengan bahasa C
  • Mampu menyiapkan lingkungan pengembangan dan menulis program C++ dari awal
  • Menguasai struktur dasar, sintaks, serta kontrol alur program seperti percabangan dan perulangan
  • Mampu menggunakan array, string, fungsi, pointer, dan reference untuk membangun program yang efisien
  • Memahami dan menerapkan konsep Pemrograman Berorientasi Objek (OOP) seperti class, object, inheritance, dan polymorphism
  • Mengelola data menggunakan struktur data dasar, file handling, serta memanfaatkan STL (Standard Template Library)
  • Menangani error dengan exception handling dan membangun kode yang modular serta mudah digunakan kembali
  • Menyelesaikan satu proyek akhir menggunakan C++ sebagai penerapan dari seluruh konsep yang telah dipelajari
